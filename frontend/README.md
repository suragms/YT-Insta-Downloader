# YouTube Downloader (Frontend Only)

This acts as a modern, responsive frontend for a YouTube Downloader.

## Setup

1.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser at `http://localhost:5173`.

## Features

-   **Modern UI**: Glassmorphism design with Tailwind CSS.
-   **Mock Functionality**: Simulates API calls to demonstrate loading states and results.
-   **Responsive**: Works on mobile and desktop.

## Project Structure

-   `src/components`: UI components (`Navbar`, `Downloader`, `ResultCard`, `Footer`).
-   `src/utils/mockApi.js`: Simulates backend responses.
